# Performance Support Website

This is a recreation of the Performance Support Website based on the provided images.

## Setup Instructions

1. **Copy Images to Assets Folder**

   You need to copy the following images from your original location to the `assets` folder with these names:

   - `banner.png` - The Performance Support Website banner (from `PerfSupport_Banner-7a7c8cd2-084f-4a33-9f18-49eafbe4e9c1.png`)
   - `new-hire.png` - New Hire Homepage button icon (from `NewHireButton-1233b81b-6098-4ad1-b5a3-9d98f478cd72.png`)
   - `knowledge-center.png` - Back to Knowledge Center button icon (from `BacktoKC-862ce189-a178-46fd-ab3b-c18457edc9a3.png`)
   - `ops-request.png` - Submit Admissions Ops Request button icon (from `OpsRequest-2ebed390-93a9-47b4-9939-bf3a189febb5.png`)
   - `search-icon.png` - Search magnifying glass icon (from `Search-Button-e49350af-caaf-4598-a735-c963d9f32cf8.png`)

2. **Open the Website**

   Simply open `index.html` in your web browser.

## Features

- **Banner**: Top banner with Performance Support Website logo and title
- **Navigation Buttons**: Three buttons on the left side for navigation
- **Content Boxes**: Multiple scrollable boxes with links organized by category
- **Search Bar**: Search functionality to filter links
- **Hover Effects**: 
  - Boxes show a shadow on hover
  - Links turn blue and underlined on hover
- **Custom Scrollbars**: Styled scrollbars for the content boxes

## Browser Compatibility

This website works best in modern browsers (Chrome, Firefox, Edge, Safari). The custom scrollbar styling works best in WebKit-based browsers (Chrome, Safari, Edge).
